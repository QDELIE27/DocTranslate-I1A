class algo.Snake(name)¶

Paramètres

name (str) – The snake’s name.

move(x, y)¶

Moves the snake to given position.

Avertissement

Be careful to not tie knots when moving the snake!

Paramètres

  • x (int) – The x position where move to.

  • y (int) – The y position where move to.

algo.translate(text)¶

Translate the given text to Parseltongue.

Paramètres

text (str) – The text to translate.

Retourne

the translated text.

Type retourné

str

>>> translate("Hello!")
'sssss!'
>>> translate("I am a sssnake!")
's ss s sssssss!'
